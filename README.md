# 💰 Personal Finance Dashboard

Take control of your finances.
Personal Finance Dashboard is a modern Android application for tracking income and expenses, managing transactions, analyzing spending habits, and monitoring personal finances.

## ✨ Features

### 💳 Transaction Management

* Add transactions
* Edit transactions
* Delete transactions
* View transaction details

### 🏷 Categories

#### Income Categories

* Salary
* Freelance
* Investment
* Gift
* Other

#### Expense Categories

* Food
* Transport
* Shopping
* Entertainment
* Health
* Education
* Bills
* Travel
* Other

### 🔍 Filtering & Sorting

Filter by:

* All
* Income
* Expense

Sort by:

* Newest
* Oldest
* Highest Amount
* Lowest Amount

### 📊 Dashboard

* Current Balance
* Total Income
* Total Expenses
* Latest Transactions

### 📈 Statistics

* Income Summary
* Expense Summary
* Category Statistics
* Monthly Statistics

### ⚙ Settings

* Dark Theme
* Light Theme
* Clear All Data

### 🎨 UI

* Material 3
* Jetpack Compose
* Responsive Layout
* Dark Mode Support

## 🏗 Architecture

* Kotlin
* MVVM
* Room Database
* StateFlow
* Coroutines
* Koin
* Navigation Compose
* Material 3
